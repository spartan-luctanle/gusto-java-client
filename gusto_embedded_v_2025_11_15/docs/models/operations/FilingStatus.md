# FilingStatus

Determines which tax return form an individual will use. One of: Single, Married, Head of Household, Exempt from withholding.

## Example Usage

```java
import com.gusto.embedded_api_v_2025_11_15.models.operations.FilingStatus;

FilingStatus value = FilingStatus.SINGLE;
```


## Values

| Name                      | Value                     |
| ------------------------- | ------------------------- |
| `SINGLE`                  | Single                    |
| `MARRIED`                 | Married                   |
| `HEAD_OF_HOUSEHOLD`       | Head of Household         |
| `EXEMPT_FROM_WITHHOLDING` | Exempt from withholding   |
